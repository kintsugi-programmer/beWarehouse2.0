Q1: Network Representation and Basic Analysis

Methods:
- Loaded the Email-Eu-core edge list file.
- Converted it into an undirected graph by storing both (u,v) and (v,u).
- Built an adjacency matrix using numpy (binary 0/1 representation).
- Computed number of nodes (N), number of edges (L), density, average degree and degree distribution.

Results:
- Nodes (N): 1005  
- Edges (L): 25571  
- Adjacency matrix shape: (1005, 1005)  
- Density: 0.050684822897464864  
- Average degree <k>: 50.88756218905473  
- Scaled degree distribution plot generated and saved.

Observations:
- The network is sparse since density << 1.
- Only a small fraction of possible edges actually exist.
- Degree distribution is right-skewed, meaning few nodes have very high degree while most are small.
- This already hints that the network is heterogeneous (not uniform).

Q3: Weighted Network Analysis

Methods:
- Loaded the OpenFlights (US airport) dataset.
- Considered it as weighted (edge weight = number of flights).
- Redefined degree using strength (sum of weights).
- Computed weighted degree distribution.
- Computed weighted clustering coefficient using Barrat formulation.
- Plotted clustering vs degree.

Results:
- Nodes loaded: 3330  
- Strength computed for all nodes.
- Weighted degree distribution plotted.
- Weighted clustering computed.
- Clustering vs degree plot generated.

Observations:
- Strength distribution is highly heterogeneous.
- A few airports dominate the total traffic.
- Clustering decreases as degree increases.
- This matches hub-and-spoke structure seen in real transport networks.
- Large hubs connect distant regions which are not necessarily connected among themselves.


Q4: Gilbert Random Graph Comparison

Methods:
- Loaded Facebook combined network (undirected, unweighted).
- Computed p for Gilbert model using same N and L.
- Generated one random instance and also averaged over 100 runs.
- Compared degree distributions (real vs random).

Results:
- Real graph nodes (N): 4039  
- Real graph edges (L): 88234  
- Computed p: 0.010819963503439287  
- Real vs random plot saved.
- Average over 100 runs plot saved.

Observations:
- Real network shows heavy-tailed / broad distribution.
- Gilbert model gives narrow binomial/Poisson-like shape.
- Even after 100 runs, random model does not resemble real one.
- This clearly shows real networks are not purely random.
- More advanced generative models are needed to capture hubs and clustering.
